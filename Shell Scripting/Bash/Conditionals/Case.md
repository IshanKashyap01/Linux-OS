# Case Statements

```bash
case $class in
    1)
        type="Samurai"
        hp="10"
        attack=20
    ;;
    2)
        type="Prisoner"
        hp="20"
        attack=4
    ;;
    3)
        type="Prophet"
        hp="30"
        attack=4
    ;;
esac
```

`case` works like a `switch` statement in java
